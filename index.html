ABOUT ME

a 1st-year Software Engineering student with a strong passion for low-level programming, algorithms, and system design. I believe that understanding how computers work at a fundamental level is the key to writing efficient, reliable, and elegant code. My theoretical background includes studying Programming: Principles and Practice Using C++ (3rd Edition) by Bjarne Stroustrup, where I mastered RAII, STL, OOP, and low-level memory management, as well as Discrete Mathematics for Programmers, covering logic, combinatorics, graph theory, and formal proofs. I’ve also dived into Theoretical Computer Science by Gromkovich, exploring automata theory, computability, and complexity, and read Code: The Hidden Language of Computer Hardware and Software by Charles Petzold to understand how code translates to hardware. My technical skills span C++, C, Python, JavaScript, HTML, and CSS, with a solid grasp of Object-Oriented Programming, procedural programming, generic programming, RAII, STL, pointers and memory management, algorithms and data structures, discrete mathematics, and theory of computation, while my toolset includes Git, VS Code, GCC/Clang, and the Linux command line. Among my projects are a ULSTU Schedule App for students to manage class schedules in real time, an Eigenvector Solver for computing eigenvalues and eigenvectors, and a Bubble Calculator — a simple yet functional tool for everyday calculations.
